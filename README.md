
## Installation & Setup

1. **Clone the repository**:
   ```
   git clone https://github.com/MommySuperior/VMHDdemo/  
   cd VMHDdemo  
   ```

2. **Create a virtual environment**:
   ```
   py -m venv .venv  
   .\.venv\Scripts\Activate.ps1
   ```

3. **Run the scripts**:
   ```
   py Reddit_PS_demo.py
   py Reddit_NC_demo.py
   py Reddit_TM_demo.py  
   ```
